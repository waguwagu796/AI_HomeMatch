# HomeMatch 프로젝트 개요서

## 📋 프로젝트 개요

**HomeMatch**는 AI 기반 주거 임대차 관리 플랫폼으로, 계약부터 입주, 거주 중 관리, 퇴실까지 전 과정을 지원하는 통합 솔루션입니다. 복잡한 법률 용어와 분쟁 위험을 AI 기술로 해결하여 사용자가 안전하고 스마트하게 주거 생활을 관리할 수 있도록 돕습니다.

---

## 🎯 기획의도 및 핵심 가치

### 문제 인식
1. **정보 불균형**: 중개사 설명과 실제 계약서 내용의 차이로 인한 피해
2. **법률 지식 부족**: 복잡한 계약서 조항을 이해하지 못해 불리한 계약 체결
3. **분쟁 예방 부재**: 입주 시점 기록 부족으로 퇴실 시 보증금 분쟁 발생
4. **관리 어려움**: 거주 중 하자 관리, 주거비 추적 등 체계적 관리 부재

### 해결 방안
- **AI 기반 계약서 분석**: 복잡한 계약서를 자동 분석하여 위험 요소 탐지
- **실시간 검증**: 중개사 설명과 계약서 내용을 실시간으로 비교 검증
- **체계적 기록 관리**: 입주부터 퇴실까지 모든 과정을 디지털로 기록
- **분쟁 예방 가이드**: 실제 분쟁 사례 기반 예방 가이드 제공

### 핵심 가치
- **투명성**: 모든 정보를 명확하게 제공
- **예방**: 분쟁 발생 전 예방적 조치
- **편의성**: 복잡한 과정을 간단하게 자동화
- **신뢰성**: 법률 근거 기반 정확한 정보 제공

## 🚀 주요 기능

### 1. 랜딩 페이지
- 서비스 소개 및 핵심 기능 안내
- 직관적인 네비게이션으로 주요 기능 접근

### 2. 회원 관리
- 이메일 기반 회원가입/로그인
- JWT 기반 인증 시스템
- 개인화된 서비스 제공

### 3. 매물 탐색
- **매물 검색**: 지역, 가격, 구조 등 필터링 검색
- **매물 상세**: 가격, 구조, 옵션, 위치 정보 조회
- **체크리스트**: 매물 확인 전 필수 체크리스트 제공

### 4. AI 계약서 분석 ⭐ 핵심 기능
- **계약서 업로드**: 이미지 파일 업로드 및 OCR 처리
- **AI 분석**: 계약서 조항을 구조화하여 분석
- **위험 요소 탐지**: 분쟁 가능한 조항 자동 탐지
- **유사 판례 제공**: 키워드 기반 유사 판례 검색 및 제공
- **추천 표준 문구**: 분쟁 최소화를 위한 수정 문구 제시
- **관련 법령 안내**: 계약 조항과 연관된 법령 정보 제공

### 5. 계약서 검증
- **중개사 설명 vs 계약서 비교**: 구두 설명과 실제 계약서 내용 대조
- **불일치 항목 탐지**: 차이점 자동 탐지 및 알림
- **위험도 평가**: 각 불일치 항목의 위험도 표시

### 6. 거주 중 관리 ⭐ 핵심 기능
- **계약 기간 관리**: 입주일/종료일 등록, 남은 기간 자동 계산
- **주거비 관리**: 
  - 월세/관리비/공과금 설정
  - 납부일 설정 및 자동 등록 옵션
  - 주거비 달력으로 시각적 관리
  - 월별 주거비 기록 및 납부 완료 처리
- **입주 상태 기록**: 
  - 입주 시점 사진 업로드 (드래그 앤 드롭)
  - 공간별 분류 (현관, 거실, 주방, 욕실 등)
  - 촬영 가이드 제공
- **거주 중 이슈 관리**:
  - 하자/이슈 사진 업로드
  - 이미지 자동 압축 (최대 1920px, 1MB 이하)
  - 처리 상태 관리 (접수 완료/처리 중/처리 완료/거절)
  - 이슈 기록 등록/수정/삭제

### 7. 퇴실 관리 ⭐ 핵심 기능
- **입주 기록 조회**: 거주 관리에서 등록한 입주 상태 기록 조회
- **퇴실 체크리스트**: 전기/가스/수도 해지, 열쇠 반납 등 필수 절차 체크
- **원상복구 체크리스트**: 바닥재, 가구, 창문, 벽지, 조명 등 확인 항목
- **퇴실 준비 일정 가이드**: D-7부터 D+14까지 단계별 가이드
- **분쟁 예방 가이드**: 
  - 분쟁 빈번 항목 상세 가이드
  - 임차인 책임 여부 안내
  - 분쟁 예방 촬영 팁 제공
- **보증금 관리**:
  - 보증금 반환 의무 안내
  - 내용증명 발송 가이드
  - 법적 조치 가이드 (지급명령, 소액소송)
  - 보증금 정보 관리

### 8. 챗봇 어시스턴트 ⭐ 핵심 기능
- **RAG 기반 답변**: 벡터 데이터베이스 기반 정확한 답변 제공
- **법령 정보 안내**: 주택임대차보호법 등 관련 법령 요약 정보
- **분쟁 조정 사례 기반**: 실제 분쟁 조정 사례를 근거로 답변
- **계약서 조항 질의응답**: 계약서 내용에 대한 질문 답변

### 9. 마이페이지
- 일정 알림 (계약 갱신, 주거지 이전)
- 회원 정보 관리
- 회원 탈퇴

## 🛠 기술 스택

### Frontend
- **Framework**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **Icons**: Lucide React
- **State Management**: React Context API

### Backend
- **Language**: Java (Spring Boot)
- **Database**: MySQL
- **Authentication**: JWT (JSON Web Token)
- **API**: RESTful API

### AI/ML
- **OCR**: 계약서 이미지 텍스트 추출
- **NLP**: 계약서 조항 분석 및 구조화
- **Vector Database**: ChromaDB
- **Embedding Model**: Sentence Transformers
- **LLM**: Ollama (Llama3.1, Gemma3:4b)
- **RAG (Retrieval-Augmented Generation)**: 분쟁 조정 사례 기반 답변 생성

### 데이터
- **분쟁 조정 사례**: 주택임대차 분쟁조정 사례집 벡터화
- **법령 데이터**: 주택임대차보호법 등 관련 법령

## 💡 차별화 포인트

### 1. AI 기반 계약서 분석
- 단순 OCR이 아닌 조항 구조화 및 위험 요소 탐지
- 유사 판례 자동 검색 및 제공
- 법령 기반 추천 표준 문구 제시

### 2. 실시간 검증 시스템
- 중개사 설명과 계약서 내용 실시간 비교
- 불일치 항목 자동 탐지 및 위험도 평가

### 3. 전주기 관리
- 계약 → 입주 → 거주 중 → 퇴실까지 전 과정 통합 관리
- 입주 시점부터 퇴실까지 모든 기록 디지털 보관

### 4. RAG 기반 챗봇
- 벡터 데이터베이스 기반 정확한 답변
- 실제 분쟁 조정 사례를 근거로 한 신뢰성 있는 정보 제공

### 5. 분쟁 예방 중심
- 사후 대응이 아닌 사전 예방에 초점
- 입주 시점 기록으로 퇴실 시 분쟁 예방
- 체계적인 가이드로 사용자 교육
