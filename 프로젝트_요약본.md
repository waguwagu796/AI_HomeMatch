# HomeMatch 프로젝트 요약

## 🎯 한 줄 소개
**AI 기반 주거 임대차 관리 플랫폼** - 계약부터 입주, 거주 중, 퇴실까지 전 과정을 지원하는 통합 솔루션

---

## 💡 핵심 가치 제안

### 문제 해결
- ❌ 복잡한 계약서 이해 어려움 → ✅ AI가 자동 분석하여 위험 요소 탐지
- ❌ 중개사 설명과 계약서 불일치 → ✅ 실시간 비교 검증
- ❌ 입주 시점 기록 부족 → ✅ 체계적 디지털 기록 관리
- ❌ 퇴실 시 보증금 분쟁 → ✅ 사전 예방 가이드 제공

---

## 🚀 핵심 기능 (5가지)

### 1️⃣ AI 계약서 분석
- 계약서 업로드 → AI 분석 → 위험 요소 탐지 → 유사 판례 제공 → 추천 표준 문구 제시

### 2️⃣ 계약서 검증
- 중개사 설명 vs 계약서 내용 실시간 비교 → 불일치 항목 자동 탐지

### 3️⃣ 거주 중 관리
- 계약 기간 관리 | 주거비 관리 | 입주 상태 기록 | 거주 중 이슈 관리

### 4️⃣ 퇴실 관리
- 퇴실 체크리스트 | 원상복구 체크리스트 | 분쟁 예방 가이드 | 보증금 관리

### 5️⃣ RAG 기반 챗봇
- 벡터 DB 기반 정확한 답변 | 분쟁 조정 사례 근거 | 법령 정보 안내

## 🛠 기술 스택

| 분야 | 기술 |
|------|------|

| **Backend** | Java (Spring Boot) + MySQL |
| **AI/ML** | ChromaDB (Vector DB) + Sentence Transformers + Ollama (LLM) |
| **핵심 기술** | RAG (Retrieval-Augmented Generation) |

## 💎 차별화 포인트

1. **AI 기반 계약서 분석** - 단순 OCR이 아닌 조항 구조화 및 위험 탐지
2. **실시간 검증** - 중개사 설명과 계약서 실시간 비교
3. **전주기 관리** - 입주부터 퇴실까지 통합 관리
4. **RAG 기반 챗봇** - 실제 분쟁 사례 기반 정확한 답변
5. **분쟁 예방 중심** - 사후 대응이 아닌 사전 예방
