# 기능정의서 추가 항목

## 9️⃣ 거주 관리 (ResidencyManagementPage)

### 거주 계약 기간 관리
- **거주 계약 기간 등록/수정**: 계약 시작일(입주일)과 종료일(거주 마감일) 등록 및 수정
- **계약 기간 자동 계산**: 시작일과 종료일을 기반으로 계약 기간(개월/일) 자동 계산
- **남은 기간 표시**: 현재 날짜 기준으로 계약 만료까지 남은 일수 표시

### 주거비 관리
- **주거비 기본 설정**: 월세, 관리비, 전기/수도/가스 예상 금액 설정
- **납부일 설정**: 매월 납부일(1~31일) 설정 및 변경
- **자동 등록 옵션**: 설정한 주거비를 매월 자동으로 기록하는 옵션
- **주거비 달력**: 월별 주거비를 달력 형태로 표시, 납부일 클릭하여 등록/수정
- **월별 주거비 기록**: 연도/월별 주거비 기록 등록, 수정, 납부 완료 처리
- **지난 주거비 내역**: 과거 월별 주거비 기록 조회

### 입주 상태 기록
- **입주 상태 사진 업로드**: 입주 시 촬영한 사진 업로드 (드래그 앤 드롭 지원)
- **공간별 분류**: 현관, 거실, 안방, 주방, 욕실, 베란다/발코니, 기타 공간 등으로 분류
- **기록 날짜 관리**: 입주 상태 기록 날짜 설정
- **입주 상태 기록 조회**: 등록된 입주 상태 기록 목록 조회
- **입주 상태 기록 삭제**: 등록된 입주 상태 기록 삭제
- **입주 상태 기록 가이드**: 입주 시 촬영해야 할 항목 가이드 제공

### 거주 중 이슈 기록
- **이슈 사진 업로드**: 거주 중 발생한 하자/이슈 사진 업로드 (드래그 앤 드롭 지원)
- **이미지 자동 압축**: 업로드한 이미지를 자동으로 리사이즈 및 압축 (최대 1920px, 약 1MB 이하)
- **이슈 제목 입력**: 이슈 이름/제목 입력
- **처리 상태 관리**: 접수 완료, 처리 중, 처리 완료, 거절 상태 관리
- **이슈 기록 등록**: 거주 중 발생한 이슈 기록 등록
- **이슈 기록 수정**: 등록된 이슈 기록 수정 (제목, 상태, 사진)
- **이슈 기록 조회**: 최근 이슈 접수 내역 조회
- **이슈 기록 삭제**: 등록된 이슈 기록 삭제

## 🔟 퇴실 관리 (MoveOutPage)

### 입주 기록 관리
- **입주 기록 조회**: 거주 관리 페이지에서 등록한 입주 상태 기록 조회 및 표시
- **입주 기록 다운로드**: 등록된 입주 상태 기록 사진 다운로드
- **입주 기록 가이드**: 입주 시 촬영 가이드 및 중요 포인트 안내

### 퇴실 체크리스트
- **퇴실 체크리스트 조회**: 퇴실 전 필수 절차 체크리스트 표시
- **체크리스트 항목 완료 처리**: 전기/가스/수도 해지, 열쇠 반납 등 항목 완료 처리
- **체크리스트 자동 저장**: 체크리스트 항목 완료 시 자동 저장
- **퇴실 준비 일정 가이드**: D-7부터 D+14까지 단계별 퇴실 준비 일정 가이드

### 원상복구 체크리스트
- **원상복구 체크리스트 조회**: 원상복구 관련 확인 항목 표시
- **체크리스트 항목 완료 처리**: 바닥재, 가구, 창문, 벽지, 조명 등 항목 완료 처리
- **체크리스트 자동 저장**: 원상복구 체크리스트 항목 완료 시 자동 저장

### 분쟁 예방 가이드
- **분쟁 빈번 항목 가이드**: 도배/장판 손상, 주방 설비 하자, 벽걸이 TV/액자 흔적 등 분쟁 빈번 항목 상세 가이드
- **임차인 책임 여부 안내**: 각 항목별로 임차인 책임이 아닌 경우와 임차인 책임이 될 수 있는 경우 구분 안내
- **분쟁 예방 팁**: 각 항목별 분쟁 예방을 위한 촬영 및 기록 팁 제공

### 보증금 관리
- **보증금 반환 의무 안내**: 임대인의 보증금 반환 의무 및 합리적 반환 기간 안내
- **내용증명 발송 가이드**: 보증금 반환 지연 시 내용증명 발송 방법 및 절차 안내
- **법적 조치 가이드**: 내용증명 발송 후에도 반환되지 않을 경우 지급명령 신청 및 소액소송 절차 안내
- **보증금 관리 정보**: 보증금 금액, 퇴실일, 반환 상태 등 보증금 관리 정보 조회 및 관리
